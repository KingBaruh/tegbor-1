## Task: Print specific letters from a string

You need to input a string `s` containing only **lowercase** letters.  
Your program should print:
- the **first** letter  converted to **uppercase**
- the **middle** letter
- and the **last** letter converted to **uppercase**

Dont use the built-in `.upper()` method to convert a character to uppercase.

**Examples**
- `s = "hello"` → `HlO`  
- `s = "world"` → `WrD`  
- `s = "abc"` → `abc`
